# MSA 환경에서 REST API, gRPC, GraphQL로 마스터하는 백엔드 API 개발
## REST API 실습
### 개발 환경
- Mac Sonoma 14
- IntelliJ IDEA 2023
- Git
- OpenJDK 17

### 실습 Spring 관련 스택
- org.springframework.boot:spring-boot-starter-web
- org.springframework.boot:spring-boot-starter-data-jpa
- org.springframework.boot:spring-boot-starter-hateoas
- org.springframework.boot:spring-boot-starter-actuator
- org.springframework.boot:spring-boot-starter-validation
- org.springframework.restdocs:spring-restdocs-mockmvc
- org.springdoc:springdoc-openapi-starter-webmvc-ui
- org.projectlombok:lombok
- com.h2database:h2


### 실습 순서
- Spring 기반 개발 환경 세팅
- 기본 REST API 구현 실습
- API 문서화 실습
- HATEOAS 구현 실습

### 실습 항목
#### Spring 기반 개발 환경 세팅
- IntelliJ – Spring  기반 프로젝트 생성 및 기본 동작 확인

#### 기본 REST API 구현 실습 (온라인 서점의 URI 설계 및 구현)
- API 엔드 포인트 및 DB 스키마 설계
- Controller 생성
- Entity 및 Repository 생성
- Controller – Service – Repository 연동
- Insomnia로 구현 API 동작 확인

#### API 문서화 실습
- Swagger 문서 작성 실습
- RestDocs로 문서 작성 실습

#### HATEOAS 구현 실습
- HATEOAS 연동 실습
